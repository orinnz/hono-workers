🚀 Hono Cloudflare App

Edge Ready

A type-safe Hono API deployed to Cloudflare Workers with integrated bindings for KV, D1, R2, Durable Objects, and Queues.

📡 API Endpoints

GET /api/health - Health check GET /api/cache/:key - Get cached value PUT /api/cache/:key - Set cached value GET /api/users - List users POST /api/users - Create user GET /api/files/:key - Download file POST /api/files/:key - Upload file GET /api/counter/:name - Get counter value POST /api/tasks - Queue a task
Running on Cloudflare Workers
